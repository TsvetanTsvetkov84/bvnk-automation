# BVNK API Test Automation

API test automation project for the BVNK QA Engineering assessment — E2E tests for
currency conversion/trade flows against the [BVNK API simulator](https://bvnkapisimulator.pythonanywhere.com/docs).

> 🚧 Work in progress — full documentation (test strategy, architecture, CI, observability) lands with the final submission.

## Quick start

```bash
yarn install
yarn test:api
```

Built with TypeScript, Playwright and Allure.
